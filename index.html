How to Apply CSS to HTML

This is how you do it:


  <!doctype html>
  <html lang="en-GB">
    <head>
      <meta charset="utf-8" />
      <title>My CSS experiment</title>
      <link rel="stylesheet" href="styles.css" />
    </head>
    <body>
      <h1>Hello World!</h1>
      <p>This is my first CSS example</p>
    </body>
  </html>
